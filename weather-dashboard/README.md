# 🌦️ Weather Dashboard Pro

A fully-featured, responsive weather dashboard application that fetches real-time weather data from OpenWeatherMap API.

## ✨ Features

### Core Features
- 🌍 **Real-time Weather Data** - Current weather conditions with detailed information
- 📍 **Location Search** - Search for any city in the world with autocomplete
- 📍 **Geolocation Support** - Get weather for your current location
- 📊 **5-Day Forecast** - Extended weather forecast with high/low temperatures
- 📈 **Hourly Forecast** - Hour-by-hour weather predictions
- 📉 **Temperature Chart** - Visual representation of temperature and humidity trends
- 💾 **Local Storage** - Save your favorite cities and settings
- ❤️ **Favorites System** - Quickly access your favorite cities
- 🎨 **Dark/Light Theme** - Toggle between dark and light modes
- ⚙️ **Customizable Settings** - Choose temperature and wind speed units
- 🔔 **Weather Alerts** - Get alerts for extreme weather conditions
- 📱 **Fully Responsive** - Works on desktop, tablet, and mobile devices

## 🚀 Quick Start

1. **Open Dashboard**: Simply open `index.html` in your web browser
2. **Get API Key**: Visit [OpenWeatherMap](https://openweathermap.org/api) and sign up for free
3. **Add API Key**: Click Settings (⚙️) and paste your OpenWeatherMap API key
4. **Search Weather**: Type a city name or click "Current Location"

## 🔑 API Information

This dashboard uses OpenWeatherMap's free API:
- **Geocoding API** - City search and autocomplete
- **Forecast API** - 5-day weather data with 3-hour intervals
- **Free Tier**: 60 calls/minute

## 💾 Data Storage

All data is stored locally in your browser:
- API Key (encrypted recommended for production)
- Temperature unit preference (°C, °F, K)
- Wind speed unit preference (m/s, km/h, mph)
- Theme preference (Dark/Light)
- Favorite cities list

## 🎨 Customization

### Temperature Units

### Wind Speed Units

## 🐛 Troubleshooting

**Weather data not loading?**
- Check internet connection
- Verify API key is correct
- Clear browser cache

**Geolocation not working?**
- Allow browser location access
- Check browser permissions
- Use HTTPS on production
